## Objectif du Projet

L’objectif de ce projet est de construire un modèle simple d’apprentissage automatique capable de classifier les fleurs d’Iris en trois espèces :

- *Iris Setosa*
- *Iris Versicolor*
- *Iris Virginica*

La classification est réalisée à partir de leurs caractéristiques mesurées, notamment :
- la longueur du sépale,
- la largeur du sépale,
- la longueur du pétale,
- la largeur du pétale.

Ce projet utilise le célèbre Iris Dataset de Fisher, largement utilisé pour l’apprentissage et la démonstration des techniques de Machine Learning.
